# Zero Vault - Secure Data Management

A full-stack application for secure data storage using zero-knowledge proofs and advanced authentication mechanisms.

## Project Structure

- `/backend`: Node.js server with SQLite/MongoDB support.
- `/zero-vault-sprint-1`: Vite-based React frontend.

## Features

- Zero-Knowledge Authentication.
- Multi-Factor Authentication (MFA).
- SQLite Database with migration support to MongoDB.
- Secure Vault storage.

## How to Run

### Prerequisities
- Node.js installed.
- MongoDB (if using the MongoDB storage provider).

### Backend Setup
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend Setup
1. `cd zero-vault-sprint-1/zero-vault-sprint-1`
2. `npm install`
3. `npm run dev`

## GitHub Upload Instructions

1. Install [Git for Windows](https://git-scm.com/download/win).
2. Open terminal in the project root.
3. Run `git init`.
4. Run `git add .`.
5. Run `git commit -m "Initial commit"`.
6. Create a new repository on [GitHub](https://github.com/new).
7. Follow the instructions on GitHub to add the remote and push.
